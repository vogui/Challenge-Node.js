remplazada, aumentada de tamaño agregando mas productos etc.

Para inicializarlo tendremos que abrir la Terminal, pararnos en la capeta, realizar el npm i para que se 
instalen las dependencias y luego realizar un npm start lo cual mostrar la  respuesta en la Terminal
